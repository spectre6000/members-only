== README

The goals of the project are as follows:

* users (manually added)
* with name, email, and password
* that can log in and out
- securely*
- with sessions/cookies that remember session state
- and remember_tokens for long term recollection
- and make posts
- that are visible to all traffic
- but only the poster is visible to other users


*very relative
